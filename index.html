Plantsy

Plantsy is my capstone project, completed as an App Academy student. In two weeks I created the first iteration of Plantsy, which is a clone of Etsy but with a plant theme. Plantsy’s first two features are item listings and the shopping cart. Additionally, I implemented some logic to randomly generate shipping dates and prices, along with a note on listings telling the user how many carts an item is in. I believe these small additions make the site feel more alive and actively used. I was excited to take on this project because it felt like the perfect opportunity to show off some of my personal design work through the seed data, in addition to my newly acquired coding skills. My future plans for this project include implementing the features: reviews, shops and transaction history.

Foxtrot

Hello, thanks for checking out our project! This is our second fullstack web application created as App Academy students. The project is a clone of Robinhood and uses Python, Flask and SQLAlchemy for the backend structure. The front end was created using JavaScript, React, Redux and vanilla CSS to style the site.

Currently the app has functionality to authenticate users, add funds to a user account, buy/sell stock, view stock details, search for a stock, view/create/delete/edit a stock watchlist, view a user portfolio, and view a user's transaction history. The stock details page shows a graph of the stock's value over time; the user portfolio page shows a graph of the value of the user's stocks over time.

Rarebnb

This is my first fullstack web application created as an App Academy student. The project is a clone of AirBnb and uses Express, Sequelize and PostgreSQL for the backend structure. The front end was created using React and Redux along with vanilla CSS to style the page.

Currently the app has functionality to authenticate users and manage spots/reviews. My future plans are to add a bookings feature which is already integrated in the backend.
